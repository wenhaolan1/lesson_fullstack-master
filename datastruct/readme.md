- 浏览器的前进 后退功能设计
    1. 不断的访问网页， 并且入栈留下记录， 方便 后退， 栈底
        后退按钮不能点， 栈空的时候  LIFO
    2. 两个栈（x,y）， 方便实现前进 后退功能
        任何一个栈为空， 标志前进或后退功能不能用
    3. 后退的时候（x 出栈）， 为了 等下可以前进， x 出栈， 在y 中入栈
        前进的时候， y 出栈， x中入栈
        看到的页面是x 的栈顶
        前进的页面是y 的战顶
        后退？ x出栈就好
    4. 当 y 里面有数据的时候， x 入栈
        x 正常入栈，y 清空一下
